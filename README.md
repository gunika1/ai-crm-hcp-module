# AI-First CRM HCP Module

An AI-powered healthcare CRM interaction logging system built using React, Redux, FastAPI, LangGraph, and Groq LLM integration.

---

# Features

- Structured HCP interaction form
- Conversational AI interaction logging
- AI-powered form auto-fill
- LangGraph agent integration
- Follow-up recommendation generation
- Responsive modern UI

---

# AI Tools

1. Log Interaction Tool
2. Edit Interaction Tool
3. HCP Search Tool
4. Follow-up Recommendation Tool
5. Product Information Tool
